# PitchPerfect

This is the first project of Udacity iOS Developer Nanodegree. It allows a user to record their voice and apply one of 6 different filters/effects to it.

## Installation
`git clone https://github.com/cmorierroy/PitchPerfect.git`<br>
Written with XCode 12.0.1.

## Usage
Run it on the XCode Simulator or on your phone (through XCode).
